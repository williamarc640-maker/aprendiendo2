Ctrl + g :para mandarse a la linea que uno quiera

shift + alt + arriba/abajo : para copiar lineas

alt + arriba/abajo : para mover la linea hacia donde uno quiera

F12 : puedes ver que hace la funcion seleccionada

Ctrl + alt + arriba/abajo : para crear multiples cursores

click un lado + Alt + otro lado : haci se puede crear un cursor nuevo en distintos lugares

selecciona algo + Ctrl + shift + L : haci se puede remplazar lo seleccionado

Ctrl + k (suelta y rapido) z : activa el modo zen

Ctrl + k + c : abre para poner los comentarios solo
